# Apple-Clone-Code
<h3>Apple Website Clone-Project Description</h3>
The Apple Website Clone is a front-end web development project that replicates the design, layouts, and user interface of the official Apple Website. The main goal of this project is to practice web development skills by creating a clean, minimal, UI similar to Apple offical website.
This clone focuses on visual accuracy and user experience, wel structured sections, and layouts accross different sizes.
<br>
<h3>Key Features</h3>
<ul>
  <b>Navigation Bar - </b>Apple-style header with logo, menu items, and icons
  <b>Product Sections - </b>Showcases Apple products like iPhone, MacBook, iPad and Watch
  <b>Hover Effects & Animations - </b>Enhances intercreativity and user experience
  <b>Footer Section - </b>Includes links, copyright text, and polices
</ul>
<br>
<h3>Technology Use</h3>
<ul>
  HTML
  CSS
</ul>
<h3>Purpose of this Project</h3>
<ul>
  To understand real world website layouts
  To improve HTML and CSS skills
  To practice web design
</ul>
